# Support

Use GitHub Issues for public support requests:

```text
https://github.com/KofTwentyTwo/license-tool/issues
```

Open a bug report for reproducible failures, incorrect audit output, release
installation problems, or documentation defects. Open a feature request for new
ecosystem support, policy/reporting changes, or workflow improvements.

For security vulnerabilities, do not include details in a public issue. Follow
the private reporting instructions in [SECURITY.md](SECURITY.md).
